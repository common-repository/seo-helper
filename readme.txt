== Description ==

The SEO Helper checks your post/draft/page for basic SEO rules that you can but must not follow. This plugin is proudly provided by <a href="http://www.all-things-seo.com" target="_blank">all-things-seo.com</a>.

== Installation ==

1. Upload the folder 'seo-helper' to your server in the folder '/wp-content/plugins/'
2. Go to the WordPress control panel and find the 'Plugins' section
3. Activate the plugin 'SEO Helper'
